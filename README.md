# mini-splitwise-api
A simplified backend version of Splitwise — just the API, no frontend (no mobile app, no website). It exposes HTTP endpoints that another app could call.

## Description 
A simplified backend version of Splitwise — just the API, no frontend (no mobile app, no website). It exposes HTTP endpoints that another app could call.

## Tech stack (list)
- **Framework:** .NET 8 (Target Framework Moniker: `net8.0`)
- **Runtime / Language:** C# with nullable reference types enabled and implicit usings
- **Application Type:** ASP.NET Core Web API (Controller-based)
- **API Documentation:** Swagger / OpenAPI (`Swashbuckle.AspNetCore` 6.6.2)
- **Transport:** HTTPS
